acq400_configure_transient

Usage
usage: acq400_configure_transient.py [-h] [--pre PRE] [--post POST] [--demux DEMUX] [--clk CLK]
                                     [--trg TRG] [--sim SIM] [--trace TRACE]
                                     [--auto_soft_trigger AUTO_SOFT_TRIGGER] [--clear_counters]
                                     uuts [uuts ...]

Positional Arguments

uuts

uut pairs: m1,m2 [s1,s2 …]

Named Arguments

--pre

pre-trigger samples

Default: 0

--post

post-trigger samples

Default: 100000

--demux

embedded demux

Default: 1

--clk

int|ext|zclk|xclk,fpclk,SR,[FIN]

--trg

int|ext,rising|falling

--sim

s1[,s2,s3..] list of sites to run in simulate mode

--trace

1 : enable command tracing

--auto_soft_trigger

force soft trigger generation

Default: 0

--clear_counters

clear all counters SLOW

Default: False
