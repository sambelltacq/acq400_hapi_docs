Named Arguments

--burst_length

Burst length : -1 same as AWG, 0: no burst, >0 special [sub] length

Default: -1

--delay

auto switch on this delay

Default: 0

--trgDX

trigger DX line

Default: 0

--monitor_uut

url of capture ADC
