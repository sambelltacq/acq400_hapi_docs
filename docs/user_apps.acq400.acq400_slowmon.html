acq400_slowmon

Usage
usage: acq400_slowmon.py [-h] [--show_raw SHOW_RAW] [--show SHOW] [--pchan PCHAN] [--root ROOT]
                         [--runtime RUNTIME] [--verbose VERBOSE] [--egu EGU]
                         [--save_file SAVE_FILE] [--slowmon_fs SLOWMON_FS]
                         uuts

Positional Arguments

uuts

uuts

Named Arguments

--show_raw

show raw data, ‘d’=decimal, ‘h’=hex

--show

show data on screen (raw or egu)

Default: 1

--pchan

max channels to print

Default: 8

--root

Location to save files. Default dir is UUT name.

Default: “”

--runtime

How long to stream data for

Default: 1000000

--verbose

Prints status messages as the stream is running

Default: 0

--egu

plot egu (V vs s)

Default: 0

--save_file

store binary output to file

--slowmon_fs

set slowmon output rate if set

Outline:

slowmon.py: receive and process slowmon data
