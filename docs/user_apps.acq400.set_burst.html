set_burst

Usage
usage: set_burst.py [-h] [--rgm RGM] [--dx DX] [--gpg GPG] [--sense SENSE] [--rtm_translen RTM_TRANSLEN]
                    [--post POST] [--pre PRE] [--trg TRG] [--es_enable ES_ENABLE] [--demux DEMUX]
                    [--config_only CONFIG_ONLY] [--clear_counters CLEAR_COUNTERS]
                    uuts [uuts ...]

Positional Arguments

uuts

uut

Named Arguments

--rgm

mode OFF|RGM|RTM

Default: “RTM”

--dx

dx d0|d1|d2

Default: “d0”

--gpg

source from gpg on|off

Default: “off”

--sense

rising|falling

Default: “rising”

--rtm_translen

transient length

Default: 1234

--post

shot length

Default: 100000

--pre

pre shot length

Default: 0

--trg

shot trigger triplet

Default: “1,0,1”

--es_enable

0 disables Event Signature

Default: 1

--demux

0: do not demux

Default: 1

--config_only

1: configure RGM, do nothing else

Default: 1

--clear_counters

clear counters (slow)

Default: 0
