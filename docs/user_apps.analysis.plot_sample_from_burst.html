data

Outline:

UUT is running in Burst/RTM, eg at 50Hz.

Stream the data to host, then - pick out all the Start Of Burst ES, then store the next sample - program plots time series for channel 1

[pgm@harlosh RGM]$ nc acq1001_343 4210 | pv > burst32-50Hz-512-3.bin
