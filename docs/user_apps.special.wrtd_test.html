wrtd_test

Usage
usage: wrtd_test.py [-h] [--pre PRE] [--post POST] [--demux DEMUX] [--clk CLK] [--trg TRG]
                    [--sim SIM] [--trace TRACE] [--auto_soft_trigger AUTO_SOFT_TRIGGER]
                    [--clear_counters] [--save_data SAVE_DATA] [--plot_data PLOT_DATA]
                    [--one_plot ONE_PLOT] [--trace_upload TRACE_UPLOAD] [--channels CHANNELS]
                    [--verbose VERBOSE] [--wrtd_txi WRTD_TXI] [--wrtd_delta_ns WRTD_DELTA_NS]
                    [--sleep SLEEP]
                    uut [uut ...]

Positional Arguments

uut

uuts

Named Arguments

--pre

pre-trigger samples

Default: 0

--post

post-trigger samples

Default: 100000

--demux

embedded demux

Default: 1

--clk

int|ext|zclk|xclk,fpclk,SR,[FIN]

--trg

int|ext,rising|falling

--sim

s1[,s2,s3..] list of sites to run in simulate mode

--trace

1 : enable command tracing

--auto_soft_trigger

force soft trigger generation

Default: 0

--clear_counters

clear all counters SLOW

Default: False

--save_data

store data to specified directory, suffix {} for shot #

--plot_data

0: plot data, all channels. N: first N, -N: first N, all one plot

--one_plot

1: plot data

--trace_upload

1: verbose upload

Default: 0

--channels

comma separated channel list

Default: “()”

--verbose

Default: 0

--wrtd_txi

tx immediate ID

--wrtd_delta_ns

set delay NS

Default: “50M”

--sleep

sleep delay

Default: 10

Outline:

tests wrtd functionality.

In particular, find minimum save DELTA_NS export UUTS=”acq2106_182 acq2106_180 acq2106_181 acq2106_172”
