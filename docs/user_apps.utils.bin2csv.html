bin2csv

Usage
usage: bin2csv.py [-h] [--nchan NCHAN] [--word WORD] [--outroot OUTROOT] [--out OUT]
                  [--paste PASTE]
                  binfiles [binfiles ...]

Positional Arguments

binfiles

file[s] to convert

Named Arguments

--nchan

number of channels

Default: 1

--word

int16|int32

Default: “int16”

--outroot

output root directory

Default: “”

--out

explicit output name

Default: “”

--paste

1: paste multiple files * 1 chan

Default: 0

Outline:

bin2csv - input raw binary, output csv

Example:

./user_apps/utils/bin2csv.py --out out_dir file1 file2
