compose_remote

Usage
usage: compose_remote.py [-h] [--port PORT] [--file FILE] [--output OUTPUT] [--nrep NREP]
                         [pattern ...] uut

Positional Arguments

pattern

Pattern 5*AA 5*BB

uut

uut hostname

Named Arguments

--port

endpoint port

Default: 5000

--file

Cmd file to send

--output

Composer output options: oneshot_rearm, oneshot, continuous or a filename

--nrep

nrep

Default: “”

Outline:

Remote control to composer webapp

Examples

Load new Template:

./user_apps/web/compose_remote.py --file=CONFIGS/cycloid_scan_templatesAABB.txt --port=5001 acq1001_434

Load new and compose:

./user_apps/web/compose_remote.py --port=5001 --file=CONFIGS/cycloid_scan_templatesAABB.txt --output=oneshot_rearm '5*AA 5*BB' acq1001_434
